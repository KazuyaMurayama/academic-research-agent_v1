# 検索戦略: コラーゲンペプチドサプリメントの習慣的摂取

**セッションID**: 20260314_collagen_peptide_supplement
**作成日**: 2026-03-14
**研究テーマ**: コラーゲンペプチドの習慣的摂取による効果・エビデンス・安全性・用量比較（5g vs 10g/日）

---

## 研究設問（PICO形式）

| 要素 | 内容 |
|------|------|
| **P (Population)** | 健常成人、関節疾患患者、皮膚老化が気になる成人 |
| **I (Intervention)** | コラーゲンペプチド（加水分解コラーゲン）の経口摂取（5〜10g/日） |
| **C (Comparison)** | プラセボ、非摂取群、5g/日 vs 10g/日 |
| **O (Outcome)** | 皮膚弾力性・保湿、関節痛・機能、骨密度、筋肉量、腸管健康、安全性 |
