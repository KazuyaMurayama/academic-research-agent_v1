## 1. エグゼクティブサマリー

### 結論サマリー

| 介入 | 効果量 /10 | エビデンス /10 | 実効性 /10 | **合計 /30** | 推奨度 |
|---|---|---|---|---|---|
| **マクロゴール（PEG）** | **9** | **10** | **6** | **25** | ★★★★★（第一選択） |
| **ラクツロース** | 6 | 7 | 8 | **21** | ★★★★（PEG非利用時の代替） |
| **ポリデキストロース（PDX）** | 5 | 4 | **9** | **18** | ★★★（軽症・予防的使用） |

### 主要メッセージ（3点）

1. **マクロゴールが3介入中で最もエビデンスが強く、効果量も最大**。ESPGHAN/NASPGHANガイドライン（2014）で第一選択に指定。排便回数の増加・便秘解消率ともにラクツロース・プラセボを有意に上回る（Cochrane SR, 2010/2016）。日本では2018年にモビコール®として承認（2歳以上）。

2. **ラクツロースはマクロゴールに効果量で劣るが、シロップで甘く飲みやすい**点で実効性が高い。PEGが利用できない場合の代替として国際ガイドラインが推奨。**ただし日本では保険上「高アンモニア血症」のみ適用**（小児慢性便秘は適応外）。

3. **⚠️ ポリデキストロースとマクロゴール/ラクツロースの直接比較RCTは存在しない**。PDXの小児便秘エビデンスは介入研究2〜3件（非RCTを含む）で規模が小さく、薬剤との単純比較は困難。ただしOTC食品として入手容易・長期安全性は高く、軽症・予防的使用では有力な選択肢。

## 2. 研究背景と目的

### 2.1 本レポートの位置付け

[R019（小児便秘・食物繊維4種比較）](2026-03-30_pediatric-constipation-fiber.md)は食品素材（ポリデキストロース・難消化性デキストリン・セルロース・オートミール）を中心に比較した。R019ではマクロゴール（PEG）は「ガイドラインの第一選択薬」として1行言及、ラクツロースは比較対照として1試験に登場するのみで、**薬剤と食品素材を同じ評価軸で横断比較した内容は存在しない**。

本レポートはその空白を埋める目的で、**処方薬2種（マクロゴール・ラクツロース）と食品素材1種（ポリデキストロース）を同一スコアリング枠組みで比較**する。

### 2.2 小児機能性便秘の概要

- **有病率**: 小児の約9〜29%（世界）。日本では保育園・幼稚園児の約20〜30%
- **定義（ROME IV基準）**: 4歳以上で週2回以下の排便、または週1回以上の便漏れ、過度な排便努力、硬い・大きな便、直腸内便塊等が1ヶ月以上続く
- **自然経過**: 治療なしでは約50%が成人まで持続。早期介入が重要
- **治療のステップ**: 食事・生活習慣改善 → 食物繊維補充（PDX等）→ 浸透圧性下剤（PEG / ラクツロース）→ 刺激性下剤（センナ等、短期のみ）

### 2.3 本レポートで比較する3介入の基本情報

| 介入 | 分類 | 主な機序 | 日本での入手 |
|---|---|---|---|
| ポリデキストロース（PDX） | 水溶性食物繊維（食品素材） | 腸内発酵→大腸水分保持・腸内環境改善 | OTC食品・Amazonで購入可 |
| マクロゴール（PEG 3350/4000） | 浸透圧性下剤（処方薬） | 腸管内水分保持→便を軟化・容量増加 | 処方箋必要（モビコール®, 2歳以上） |
| ラクツロース | 浸透圧性下剤 + 腸内環境改善（処方薬） | 腸管内osmotic効果 + 腸内細菌叢改善 | 処方箋必要（モニラック®等） |

## 3. 評価方法論（3軸スコアリング基準）

各介入を以下の3軸・各10点満点（計30点）でスコアリングする。**各軸は独立して評価し、合計点で総合推奨度を判定**する。

### 軸1: 効果量（10点満点）

| 指標 | 配点 | 基準 |
|---|---|---|
| 排便回数の増加（回/週） | 4点 | +3.0以上=4 / +2.0〜2.9=3 / +1.0〜1.9=2 / +0.5〜0.9=1 / 0.5未満=0 |
| 便コンシステンシー改善 | 3点 | Bristol Stool Scale改善: 明確=3 / 中程度=2 / 限定的=1 / なし=0 |
| 治療成功率（responder rate） | 3点 | ≥70%=3 / 50〜69%=2 / 30〜49%=1 / 30%未満=0 |

### 軸2: エビデンス強度（10点満点）— GRADE準拠

| 指標 | 配点 | 基準 |
|---|---|---|
| 小児対象RCT本数 | 3点 | 5本以上=3 / 2〜4本=2 / 1本=1 / 0本=0 |
| Cochrane SR または高質MAの有無 | 3点 | Cochrane SR=3 / 一般的SR/MA=2 / 個別RCT引用のみ=1 / なし=0 |
| ガイドライン推奨度 | 2点 | 国際ガイドライン第一選択=2 / 代替推奨=1 / 言及なし/推奨外=0 |
| 試験規模・追跡期間の十分性 | 2点 | n>200かつ追跡≥12週の試験あり=2 / 中規模・短期=1 / 小規模のみ=0 |

### 軸3: 実効性（10点満点）

**定義**: 入手経路の容易さ＋経済的負担の少なさ＋服用継続しやすさ＋副作用の少なさ＋長期安全性データの厚み

| 指標 | 配点 | 基準 |
|---|---|---|
| 入手経路 | 2点 | OTC食品・薬局自由購入=2 / 処方箋（一般科）=1 / 専門医処方のみ=0 |
| コスト（月額・保険外自己負担） | 2点 | 1,000円未満=2 / 1,000〜3,000円=1 / 3,000円以上=0 |
| 服用しやすさ（味・形状・用量調整） | 2点 | 食品に溶ける・甘い・用量調整容易=2 / やや飲みにくい=1 / 飲みにくい・量が多い=0 |
| 副作用の少なさ | 2点 | 重篤副作用なし・軽度副作用<10%=2 / 軽度10〜30%=1 / 30%以上または注意すべき副作用あり=0 |
| 長期安全性データ | 2点 | 長期RCTあり（≥6ヶ月）=2 / 短期のみ=1 / データなし=0 |

### 総合推奨度の判定基準

| 合計点 | 推奨度 | 意味 |
|---|---|---|
| 26〜30点 | ★★★★★ | 強く推奨（第一選択） |
| 21〜25点 | ★★★★ | 推奨（代替選択肢として有力） |
| 16〜20点 | ★★★ | 条件付き推奨（軽症・特定状況で有効） |
| 11〜15点 | ★★ | 限定的推奨（エビデンス不十分） |
| 〜10点 | ★ | 非推奨 |
