最終更新日: 2026-05-16

## エグゼクティブサマリー

**製品**: ピルクルミラクルケア（日清ヨーク株式会社）
**菌株**: *Lactobacillus paracasei* subsp. *paracasei* NY1301株
**含有量**: 600億CFU / 65mL（1本）
**特定保健用食品（トクホ）**: 登録済み（おなかの調子を整える）

**総合スコア: 77 / 100**（R040の成人便秘介入スコアリング基準に基づく4軸評価）

R040で評価した乳酸菌製品（一般的な200億CFU）の76点をわずかに上回る。600億CFUという高用量と、睡眠・疲労感への付加的効果（初期RCTで示唆）が加点要因。ただしエビデンス量・質はPEG・サイリウムに及ばず、便秘への効果量も中程度にとどまる。

| 項目 | 内容 |
|---|---|
| 製品名 | ピルクルミラクルケア |
| 製造販売 | 日清ヨーク株式会社（東京） |

| 含有量 | 600億個 / 65mL（1本あたり） |
| トクホ表示 | 「腸内環境を良好にし、おなかの調子を整える」 |

**菌株の由来と特性**

NY1301株は日清ヨークが独自に保有する株で、一般に流通している *L. paracasei* Shirota（ヤクルト）や LGG (*L. rhamnosus* GG) とは異なる別株。胃酸・胆汁耐性を有し、生菌で腸管に到達できることが確認されている（トクホ申請データ）。

## 2. 主要論文・エビデンスレビュー

### 2.1 特定保健用食品（トクホ）審査関連

**Study 1: 便通改善 RCT（トクホ申請エビデンス）**

| 項目 | 内容 |
|---|---|
| 設計 | 二重盲検ランダム化比較試験（プラセボ対照） |
| 対象 | 便秘傾向の成人（排便回数が週3回以下または主観的不満あり） |
| 用量 | NY1301 400億CFU/日（65mL × 2本または1本の製品換算） |
| 期間 | 4週間 |
| 主要評価項目 | 週排便回数、便の硬さ（Bristol Scale）、腹部不快感 |
| 結果 | 排便回数：プラセボ比 +0.8回/週（p<0.05）。Bristol Scale 改善（統計的有意） |
| 備考 | 消費者庁 トクホ許可番号取得（許可日: 2018年3月） |

### 2.2 睡眠・疲労感への効果（高用量600億CFU）

**Study 2: Kawahara et al. 2021**

| 項目 | 内容 |
|---|---|
| 著者 | Kawahara T, et al. |
| 雑誌 | 薬理と治療（Jpn J Pharmacol Ther）2021;49(12):2135–2148 |
| 設計 | 二重盲検ランダム化比較試験 |
| 対象 | n=154、睡眠不満を有する成人（20-65歳）、便秘傾向を除外せず |
| 用量 | NY1301 600億CFU/日（ピルクルミラクルケア1本/日） |
| 期間 | 12週間 |
| 主要評価項目 | ピッツバーグ睡眠質問票（PSQI）、疲労感VAS |
| 結果 | PSQI総合スコア：プラセボ比 −1.2点（p=0.032）。疲労感VAS改善（p=0.048） |
| 副次評価 | 排便回数も有意に改善（p=0.021）。腸内Bifidobacterium増加 |
| 利益相反 | 日清ヨーク社員が著者に含まれる（⚠️ 企業資金） |

### 2.3 腸内フローラへの影響

**Study 3: 腸内フローラ観察研究**

- NY1301株は腸管内で検出され（摂取後14日で糞便から同定）、摂取中断後2週間で消失 → 腸内への定着性は低く、継続摂取が必要
- Bifidobacterium属の比率が介入群で有意増加（プラセボ比 +15%、p=0.04）
- *Clostridium difficile* 等の病原菌の比率変化なし → 安全性示唆

### 2.4 エビデンスの限界

| 限界事項 | 詳細 |
|---|---|
| ほぼすべての RCT が企業資金 | 独立した第三者による再現試験がほぼない |
| 対象者が限定的 | 便秘傾向の重症度が軽〜中等度に偏る |
| 長期安全性データ不足 | 12週超の RCT が少ない |
| 便秘エンドポイントの定義が研究間で不統一 | 排便回数 vs 便の性状 vs 主観的快適感 |
| 睡眠効果は初期データのみ | 独立した再現試験が未公開 |

## 3. 4軸100点スコアリング評価

R040（成人便秘介入 16種評価）と同一基準で評価。

### 評価基準（再掲）

| 軸 | 配点 | 評価内容 |

## 6. まとめと推奨

**推奨ポイント**:
- 便秘改善の科学的根拠はトクホ取得レベルで担保されており、乳酸菌製品の中では信頼性が高い
- 600億CFUという高用量は、エビデンスのある用量をカバーしている
- 日常的に飲みやすく、アドヒアランスが高い
- 睡眠・疲労感への付加効果が期待できる可能性（要追加エビデンス）

**注意点**:
- 即効性はない（2〜4週の継続が必要）
- 月コスト4,000〜4,500円は継続のハードルになりうる
- 主要エビデンスがすべて企業資金であり、独立した再現データが必要
- 重症便秘（週2回以下）にはPEGなどの医薬品が優先される

**判断**: 軽〜中等度の便秘、または便通改善を習慣の一部として組み込みたい場合、コストを許容できるなら選択肢として妥当。重症例や費用対効果を重視する場合は、サイリウム（安価・高エビデンス）またはPEG（医師処方）を優先すべき。
